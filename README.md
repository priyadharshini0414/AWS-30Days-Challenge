##  AWS Hands-on Tasks Performed

Throughout this challenge, I worked on real-time AWS scenarios including:

###  Compute & Storage
- Launched and configured EC2 instances
- Created and attached EBS volumes
- Created AMIs and managed instance lifecycle
- Configured Elastic IPs

###  Networking (VPC)
- Created VPC, Subnets (Public & Private)
- Configured Route Tables
- Set up Internet Gateway
- Implemented NAT Instance for private subnet internet access
- Established VPC Peering between networks

### Security & Access Management (IAM)
- Created IAM Users & Groups
- Attached Policies and Roles
- Implemented least privilege access
- Enabled secure SSH access to EC2

###  Load Balancing & Monitoring
- Configured Application Load Balancer
- Set up CloudWatch alarms
- Enabled instance monitoring

###  Storage & Backup
- Created EBS snapshots
- Managed data persistence

###  Web Server Setup
- Installed and configured Nginx on EC2
- Hosted web application

###  Advanced Networking
- Configured public & private architecture
- Ensured secure communication between instances
